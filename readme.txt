How to install

1. Run 1.12.2 then close the game and the launcher
2. open file manager and look on the top bar for view
3. Click hidden items so its checked on
4. Double click the forge install file and install client
5. Hit windows + r
6. type %appdata% then enter
7. Open the roaming folder if it exists
8. drag the mods folder from this into the .minecraft folder
